This folder contains original TIF files from FlashGordon and .rawtraces extracted using gettraces in batch mode. Example:

V2Rpp_10uM_000.rawtraces
V2Rpp_10uM_000.tif
V2Rpp_10uM_001.rawtraces
V2Rpp_10uM_001.tif
V2Rpp_10uM_002.rawtraces
V2Rpp_10uM_002.tif
V2Rpp_10uM_003.rawtraces
V2Rpp_10uM_003.tif
...

After that, run `selectPrintedSpots` and save output `.rawtraces` into corresponding A, B, C, etc folders.
